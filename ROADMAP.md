# Roadmap inicial

Fases propuestas:
1. Especificación del formato de archivo (v0.1) — objetivo: definir campos, validaciones y ejemplos.
2. Ejemplos y librería de componentes básicos — resistores, condensadores, fuentes y componentes lógicos.
3. Editor básico web y CLI (crear/editar/validar) — UI mínima para dibujar y JSON para guardar.
4. Adaptadores de simulación (SPICE, Verilog) — export/import de formatos comunes.
5. Sistema de plugins y marketplace de componentes.
6. Integraciones con IDEs (extensión para VS Code) y plataformas colaborativas.

Cada fase se dividirá en issues y milestones; puedo crear esas issues automáticamente si quieres.
